# Проект работы с виджетами банковских операций клиента.
* Модуль **masks** содержит функции:
-   get_mask_card_number -> Возвращает маску по номеру карты,
-   get_mask_account     -> Возвращает маску счета.
* Модуль **processing** содержит функции:
-   filter_by_state -> Фильтрует список словарей по ключу "state"
-   sort_by_date    -> Сортирует список словарей по ключу "date"
* Модуль **widget** содержит функции:
-   mask_account_card -> Маскирует названия карты или счета,
-   get_date          -> Выводит дату в формате: ДД.ММ.ГГГГ
* Модуль **generators** содержит функции:
-   filter_by_currency       -> Функция-генератор возвращает транзакции заданной валюты,
-   transaction_descriptions -> Функция-генератор возвращает описание операции
-   card_number_generator    -> Функция-генератор номеров банковских карт
* Модуль **decorator** содержит функцию:
-   log  -> Декоратор для логирования результов и ошибок в консоль или в файл
* Модуль **external_api** содержит функцию:
-   transaction_amount_in_rub -> Конвертация суммы транзакции в рубли с использованием ресурса api.apilayer.com

# Пример запуска финкции *mask_account_card* в модуле *widget*:
```
if __name__ == "__main__":
    print(mask_account_card('Visa Gold 599899414228426353'))
```
